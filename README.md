📞 BlockRing

BlockRing — мой первый полноценный Android-проект, разработанный на Kotlin.

Приложение автоматически блокирует входящие звонки с номеров, которых нет в контактах пользователя.

------------------------------------------------------------------------------------------------------------------

✨ Возможности

- 🚫 Автоматическая блокировка неизвестных номеров
- 👥 Просмотр контактов
- 👁️ Скрытие и восстановление контактов
- 📋 Просмотр списка заблокированных номеров
- 📌 Копирование заблокированных номеров
- 🔄 Автоматическое удаление номера из списка заблокированных после его сохранения в контакты
- 🟢 Включение и отключение защиты

------------------------------------------------------------------------------------------------------------------

🛠 Используемые технологии

- Kotlin
- Jetpack Compose
- Android SDK
- CallScreeningService
- SharedPreferences

------------------------------------------------------------------------------------------------------------------

⚙️ Как работает приложение

При входящем звонке приложение проверяет номер телефона:

- ✅ Если номер есть в контактах — звонок разрешается.
- 🚫 Если номера нет в контактах — звонок автоматически отклоняется.
- 📋 Заблокированный номер сохраняется в разделе **«Заблокировано»**.

------------------------------------------------------------------------------------------------------------------

📦 Установка

1. Установите APK-файл на устройство.
2. Разрешите доступ к контактам.
3. Выберите **BlockRing** в качестве приложения для определения номера и защиты от спама.
4. Включите защиту внутри приложения.
